README
=======

The application is written in python. The application will run independent of the operating system.

The system is expected to have python and mysql installed.

Dependencies for python can be found in requirements.txt
 that can be install using `pip install -r requirements.txt`

The data can be inserted into mysql server using `python insert.py`
The application can be started by running `python Main.py`, then can be
accessed in the browser at http://localhost:5000
